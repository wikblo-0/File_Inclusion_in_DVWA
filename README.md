This repository stores the payloads used in the penetration testing of the File Inclusion module in Damn Vulnerable Web Application (DVWA).

* lfi_low_security_level.txt stores the individual payloads used for local file inclusion on the low security level.

* rfi_low_security_level.txt stores the individual payloads used for remote file inclusion on the low security level.

* rfi_low_security_level.php stores the php file contents used for remote file inclusion on the low security level.

* lfi_medium_security_level.txt stores the individual payloads used for local file inclusion on the medium security level.

* rfi_medium_security_level.txt stores the individual payloads used for remote file inclusion on the medium security level.

* rfi_medium_security_level.php stores the php file contents used for remote file inclusion on the medium security level.

* lfi_high_security_level.txt stores the individual payloads used for local file inclusion on the high security level.



You can read more about the attack in the blog here: https://pentesting-dvwa.hashnode.dev/
